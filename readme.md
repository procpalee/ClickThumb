# 🎨 나만의 썸네일 생성기 (Thumbnail Maker) 사용 설명서

설치 없이 브라우저에서 바로 실행 가능한 **'나만의 썸네일 생성기'** 사용법을 소개합니다.
복잡한 디자인 툴 없이도, 몇 번의 클릭만으로 블로그와 유튜브에 최적화된 고퀄리티 썸네일을 만들 수 있습니다.

## 제작 동기

워드프레스 블로그와 네이버 블로그 운영을 시작하면서 썸네일의 중요성을 알게 되었습니다. 다만 이를 매번 만드는 것은 번거로운 일이었습니다.

미리캔버스와 피그마 같은 툴을 사용하더라도 의외로 많은 시간이 소요 되었고 "내용이 중요한데, 껍데기 만드는 데 힘을 빼고 있구나"라는 생각이 들었습니다.

나노바나나와 같은 생성형 AI에게 만들어 달라고 할 수도 있지만 매번 썸네일을 요청할 때마다 토큰이 소모된다는 점이 부담스럽고, 시리즈나 카테고리별로 일관된 톤앤 매너를 유지해야 하는데 생성형 AI의 특성상 한계가 있었습니다.

또한 썸네일을 자동으로 만들어 준다는 서비스가 존재하긴 했지만 디자인적으로 너무 획일화 되어 있고 제가 원하는 기능을 갖추고 있지 않아 **직접 제작**하게 되었습니다.

## 제작 동기

- **설치 없는 간편함**: HTML 파일 하나만 열면 인터넷 없이도 바로 실행됩니다.
- **최신 디자인 트렌드 반영**: 깔끔한 **Pretendard 폰트**와 세련된 **글래스모피즘(Glassmorphism)** 스타일이 기본 적용되어 있습니다.
- **비율 제어**:
    - **배경 비율**: 유튜브(16:9), 인스타(1:1) 등 플랫폼에 맞춰 배경 크기를 조절합니다.
    - **Safe Zone**: 배경과 별개로 **텍스트가 배치될 영역의 비율**을 따로 설정하여 안정적인 레이아웃을 만듭니다.
- **디테일한 커스텀**:
    - **폰트 & 타이포**: 본문/제목용 폰트 3종(Pretendard, Noto Sans, 고운바탕)과 굵기/크기를 자유롭게 조절합니다.
    - **배경 & 오버레이**: 그라디언트 색상은 물론, 이미지를 넣고 **어둡기(Overlay)**를 조절해 글씨를 잘 보이게 할 수 있습니다.
- **쾌적한 사용성**:
    - **실시간 미리보기**: 입력하는 즉시 0.1초의 지연도 없이 디자인이 화면에 반영됩니다.
    - **철저한 보안**: 서버로 데이터가 전송되지 않고 내 컴퓨터(브라우저)에서만 작동하여 안전합니다.
- **고화질 내보내기**: **WebP** 포맷과 **4배(4x)** 스케일 업을 지원하여, 대형 모니터에서도 선명한 썸네일을 만듭니다.

## 사용 방법 (Step-by-Step)

### 1. 프로그램 실행
다운로드 받은 `thumbnail_maker.html` 파일을 더블 클릭합니다. (기본 1280x720 해상도, 깔끔한 흰색 강조선으로 시작됩니다)

### 2. 비율 설정 (Size & Layout)
가장 먼저 어떤 플랫폼에 올릴지 비율을 정하세요.
- **비율 (Background Ratio)**: 유튜브라면 **16:9**, 인스타 피드라면 **1:1**을 선택하세요. 세로 폭을 직접 수정하면 **'사용자 지정'** 모드가 됩니다.
- **콘텐츠 비율 (Safe Zone)**: 텍스트가 중앙에 예쁘게 모이도록 안전 영역을 설정합니다. **'1:1'**로 설정하면 썸네일 중앙 정사각 박스 안에 텍스트가 줄바꿈되어 들어갑니다.

### 3. 텍스트 입력 (Content Controls)
왼쪽의 텍스트 입력란에 내용을 적어주세요.
- **태그 (Tag)**: 블로그 카테고리나 시리즈 명 (예: REVIEW, TIP)
- **제목**: 핵심 메시지 (입력하는 대로 실시간 반응하며, Safe Zone을 넘어가면 자동 줄바꿈 됩니다)
- **부제**: 보충 설명

### 4. 스타일 꾸미기 (Typography & Colors)
- **폰트/크기/굵기**: 태그, 제목, 부제 각각의 스타일을 미세하게 조정하여 위계(Hierarchy)를 잡아보세요.
- **색상**: 배경에 어울리는 텍스트 색상을 고르고, **강조선(Line)** 색상으로 포인트를 주세요. (기본값: 흰색)

### 5. 배경 꾸미기 (Background)
- **그라디언트**: **'시작 색상'**과 **'끝 색상'**을 각각 골라 오묘한 빛깔을 만들어보세요.
- **이미지**: 사진을 업로드하고 **[배경 어둡기]**를 조절해 텍스트 가독성을 높이세요.

### 6. 저장하기 (Export)
하단의 **[이미지 다운로드]** 버튼을 클릭하여 완성된 썸네일을 저장하세요.
- *Tip*: 블로그/웹용이라면 **WebP**, 투명 배경이 필요하면 **PNG**를 추천합니다.

## 💡 활용 팁
- **만능 비율 (16:9 + 1:1)**: 배경 비율을 **16:9**로, 콘텐츠 비율을 **1:1**로 설정해보세요. 이렇게 만든 썸네일은 유튜브/블로그 헤더(가로형)로도 쓰고, 가운데만 잘라서 인스타그램/카카오톡(정사각)으로 공유해도 텍스트가 잘리지 않는 **'만능 썸네일'**이 됩니다.
- **시리즈별 컬러링**: 시리즈나 주제별로 고유한 **태그 색상**과 **그라디언트 조합**을 정해두면, 독자들이 썸네일 색깔만 보고도 주제를 파악할 수 있습니다.

이제 직접 만들어보세요! 여러분의 콘텐츠가 더욱 돋보이게 될 것입니다.

## 🌐 웹 배포 가이드 (GitHub Pages)

이 프로젝트는 정적 웹사이트(HTML/CSS/JS)이므로 GitHub Pages를 통해 **무료로 웹에 배포**할 수 있습니다.

### 배포 순서
1. **GitHub 저장소 생성**: GitHub에 로그인 후 `New repository`를 클릭해 새 저장소를 만듭니다.
2. **코드 업로드**:
   ```bash
   git remote add origin https://github.com/[사용자명]/[저장소명].git
   git branch -M main
   git push -u origin main
   ```
3. **Pages 활성화**:
   - 저장소의 **Settings** 탭으로 이동
   - 왼쪽 메뉴에서 **Pages** 클릭
   - **Build and deployment** > **Branch** 설정에서 `main` 브랜치 선택 후 **Save**
4. **접속 확인**:
   - 약 1~5분 후 `https://[사용자명].github.io/[저장소명]/` 주소로 접속하면 웹 앱이 실행됩니다.
